# 月次家計振り返り

毎月の家計データ（マネーフォワード）をPDF保存し、分析してNotionにレポートを投稿するスキル。

## 実行手順

### Step 1: 対象月の特定
- 今日の日付から前月を対象月として特定する（例: 4月5日に実行 → 3月分）
- 対象月を `YYYYMM` 形式で変数として保持する（例: `202603`）

### Step 2: PDF保存の確認・ガイド

以下の2つのPDFが既に保存されているか確認する:
- `~/Library/Mobile Documents/com~apple~CloudDocs/02_家計_資産/03_収支内訳/{YYYYMM}_収支.pdf`
- `~/Library/Mobile Documents/com~apple~CloudDocs/02_家計_資産/04_入出金歴/{YYYYMM}_入出金.pdf`

**両方存在する場合:** 「PDF確認済み」と伝えてStep 3へ進む。

**存在しない場合:** ユーザーに以下を依頼する:

> マネーフォワードからPDFを保存してください:
>
> 1. **収支内訳**: https://moneyforward.com/cf → 対象月を選択 → ブラウザの印刷機能(⌘P)でPDFとして保存
>    - 保存先: `iCloud/02_家計_資産/03_収支内訳/{YYYYMM}_収支.pdf`
> 2. **入出金履歴**: https://moneyforward.com/cf/detail → 対象月を選択 → 同様にPDF保存
>    - 保存先: `iCloud/02_家計_資産/04_入出金歴/{YYYYMM}_入出金.pdf`
>
> 保存したら教えてください。

ユーザーから完了の連絡を待ってからStep 3へ。

### Step 3: PDFの読み取り

Read ツールで両方のPDFを読み取る:
- `{YYYYMM}_収支.pdf` — カテゴリ別の収入・支出の集計
- `{YYYYMM}_入出金.pdf` — 個別の入出金明細

### Step 4: 家計設計ルールの取得と予算基準の確定

Notionの「家計管理」ページ配下にある**最新の家計設計ルール**を取得する:
1. Notion search で「家計設計ルール」を検索し、最新版を特定
2. Notion fetch でルール全文を取得

このルールに記載されているカテゴリ別月次予算を、Step 6のレポートにおける**評価基準**として使う。
予算項目の例（ルール改訂で変わるため、必ずルール本文から読み取ること）:
- 食費、交際費、住宅（固定費）、日用品 → 日常生活費の予算
- 特別支出プール / 不可避プール → 大型支出の積立設計
- 交通費プール、ふるさと納税プール → 別枠管理

**重要:** ルールに「プール対象外」「余剰から使う」等の運用方針がある場合、それも評価に反映する。

### Step 5: 前月レポートの取得（任意）

Notionの「月次家計レポート」DB（`1d05aeb4-9fbd-8084-8ea6-de7ab32ca6da`）から前月のレポートを検索し、前月との比較や前月の課題に対する進捗を確認する。

### Step 6: レポート作成

以下の構成で月次レポートを作成する。トーンは丁寧だがフランクに。良い点は素直に褒め、課題は具体的に指摘する。

**レポート内の予算額は必ず家計設計ルールから引用し、ハードコードしない。**

```
{対象月}の家計管理の振り返りをご報告いたします。
（1-2文の総括コメント）

---

## {年}年{月}月の家計分析

### 1. 収支概要
| 項目 | 金額 |
|------|------|
| 当月収入 | ¥XXX |
| 当月支出 | ¥XXX |
| 当月収支 | ¥XXX |

（収支に対するコメント）

### 2. 支出カテゴリ別詳細（対 家計設計ルール）

日常生活費（家計設計ルールのカテゴリ別予算と比較）:
| カテゴリ | 実績額 | ルール上の予算 | 乖離額 | 評価 |
|----------|--------|---------------|--------|------|

- 「大型支出の判断基準」等の運用ルールは守れているか？

（具体的なアクションアイテムと総まとめ）
```

### Step 7: Notionに投稿

以下の情報でNotionの月次家計レポートDBに新規ページを作成する:
- **parent**: data_source_id `1d05aeb4-9fbd-8002-bb4b-000b43c70860`
- **properties**:
  - `名前`: `{YYYYMM}`（例: `202603`）
  - `収入`: 収入合計（数値）
  - `支出`: 支出合計（数値）
- **content**: Step 6で作成したレポート本文

投稿後、NotionのページURLをユーザーに共有する。

### Step 8: 完了報告

「{YYYYMM}の月次家計レポートをNotionに投稿しました → {URL}」と報告して終了。
